# Consolidated Scoring Systems Reference

Quick-reference for all validated scoring systems used across trauma, EGS, and SCC domains.

## Critical Care

| Score        | Purpose                           | Key Threshold                             |
| ------------ | --------------------------------- | ----------------------------------------- |
| APACHE II/IV | ICU mortality prediction          | Higher = worse prognosis                  |
| SOFA         | Organ failure trajectory          | >=2 increase from baseline = sepsis       |
| qSOFA        | Quick sepsis screening (ED/floor) | >=2 (RR>=22, SBP<=100, altered mentation) |
| RASS         | Sedation target                   | -5 to +4, target 0 to -2 per PADIS        |
| CAM-ICU      | Delirium screening                | Positive = delirium (assess q shift)      |
| CPOT         | Pain (non-verbal patients)        | 0-8, >=3 = significant pain               |
| BPS          | Pain (non-verbal patients)        | 3-12, >5 = significant pain               |
| NUTRIC       | ICU nutrition risk                | >=5 = high nutrition risk                 |
| Caprini      | VTE risk assessment               | 0-2 low, 3-4 moderate, >=5 high           |
| KDIGO AKI    | Renal failure staging             | Stage 1-3 by creatinine/UOP               |

## Trauma

| Score     | Purpose              | Key Threshold                        |
| --------- | -------------------- | ------------------------------------ |
| ISS       | Injury severity      | >=16 = major trauma                  |
| GCS       | Neurologic status    | <=8 = severe TBI                     |
| RTS       | Triage and outcome   | 0-7.84                               |
| TRISS     | Survival probability | Combines RTS + ISS + age + mechanism |
| AAST OIS  | Organ injury grading | Grade I-V per organ                  |
| ABC Score | MTP activation       | >=2 = activate MTP                   |
| FAST      | Free fluid detection | Positive = operative consideration   |

## EGS

| Score               | Condition                 | Key Threshold                               |
| ------------------- | ------------------------- | ------------------------------------------- |
| AAST EGS Grade      | Multiple EGS conditions   | Grade I-V anatomic severity                 |
| Alvarado (MANTRELS) | Appendicitis              | >=7 = high probability                      |
| RIPASA              | Appendicitis (Asia)       | Alternative to Alvarado                     |
| Hinchey             | Diverticulitis            | I-IV (guides operative management)          |
| Tokyo TG18          | Cholecystitis/Cholangitis | Grade I (mild), II (moderate), III (severe) |
| Atlanta Revised     | Pancreatitis              | Mild, moderately severe, severe             |
| LRINEC              | NSTI                      | >=6 suspicious, >=8 strongly predictive     |
| Ranson              | Pancreatitis              | >=3 = severe                                |
| BISAP               | Pancreatitis              | >=3 = increased mortality                   |
| Boey                | Perforated PUD            | 0-3 (illness, shock, >24h perforation)      |
| MPI                 | Peritonitis               | >=26 = high mortality                       |

## Sepsis

| Score | Purpose            | Key Threshold                    |
| ----- | ------------------ | -------------------------------- |
| SOFA  | Sepsis criteria    | >=2 point increase from baseline |
| qSOFA | ED screening       | >=2 = high risk                  |
| NEWS2 | Ward deterioration | >=7 = critical                   |

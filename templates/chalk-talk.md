## APPENDIX: Instructional Design Framework

### Gagne's 9 Events of Instruction — Mapped to Chalk Talk

| Gagne's Event                         | Chalk Talk Section                          | Implementation                               |
| ------------------------------------- | ------------------------------------------- | -------------------------------------------- |
| 1. Gain attention                     | Session Setup → Topic Announcement          | Clinical context hook, recent case relevance |
| 2. Inform learners of objectives      | Session Setup → Learning Objectives         | Explicit 3-4 objectives                      |
| 3. Stimulate recall of prior learning | Pre-Assessment Question                     | Baseline knowledge poll, activate schema     |
| 4. Present content                    | Core Teaching → 8 Talking Points            | Structured content delivery with visual aids |
| 5. Provide learning guidance          | Board Drawings, Clinical Pearls             | Visual scaffolding, memorable anchors        |
| 6. Elicit performance                 | Audience Interaction, Interactive Case      | Active participation, decision-making        |
| 7. Provide feedback                   | Interactive Case → Reveal sections          | Immediate evidence-based feedback            |
| 8. Assess performance                 | Post-Assessment Question                    | Parallel scenario to measure growth          |
| 9. Enhance retention/transfer         | Clinical Pearls, Mnemonics, Common Mistakes | Concrete takeaways for bedside application   |

### ADDIE Phases for Chalk Talk Development

**Analyze:**

- Audience: [Skill level, prior knowledge, learning gaps]
- Context: [Bedside vs. conference, time constraints, clinical service needs]
- Resources: [Whiteboard/chalkboard availability, case access, evidence base]

**Design:**

- Learning objectives (see Section 1)
- Assessment strategy (pre/post questions)
- Content sequencing (8 Talking Points framework)

**Develop:**

- Create board drawing templates
- Prepare interactive case with decision points
- Curate clinical pearls and mnemonics

**Implement:**

- Deliver chalk talk (45 min total)
- Adapt pacing based on audience engagement
- Document real-time questions for future iterations

**Evaluate:**

- Post-assessment performance vs. pre-assessment
- Audience feedback (informal or 1-minute paper)
- Self-reflection: what worked, what to revise

### Tips for Adapting Difficulty Level

**For Junior Learners (Interns, MS3/MS4):**

- Focus on Talking Points 1-5 (scenario → diagnosis → initial management)
- Use closed-ended audience interaction (multiple choice, yes/no)
- Pre-draw complex diagrams (less time on rendering, more on explanation)
- Provide more explicit mnemonics and checklists

**For Senior Learners (Senior Residents, Fellows):**

- Emphasize Talking Points 6-8 (complications, special populations, controversies)
- Use open-ended interaction ("What would you do here?" without options)
- Draw diagrams collaboratively (invite learners to board)
- Challenge with edge cases and gray-zone evidence discussions

**For Mixed Audiences:**

- Layer content: start with foundational concepts, build to nuance
- Use tiered questions: "Junior team, what's your first step?" → "Senior team, what are you worried about?"
- Acknowledge uncertainty: "This is debated — here's what the guidelines say, here's what we do, here's why"

# /asf:draft-paper

Full evidence-to-manuscript pipeline.

## Usage

```
/asf:draft-paper {topic} [--journal JTACS|Annals|JAMA-Surg|WJS|JSR|BJS|Surgery|CCM]
```

## Workflow

1. Run evidence synthesis (Module 1 + domain module)
2. Select journal profile and reporting checklist
3. Run 5-stage IMRAD pipeline (OUTLINE, DRAFT, VALIDATE, POLISH, SUBMIT-PREP)
4. Validate: >=90% checklist compliance
5. Optional: Run naturalization before submission

## Output

IMRAD manuscript, compliance report, cover letter, submission package.

# /asf:capacity-assessment

Global surgery research protocol design.

## Usage

```
/asf:capacity-assessment {country or region} [--framework lancet|nsoap|zamsat|who-eesc]
```

## Workflow

1. Select framework and research type
2. Load country/region data (WHO region, World Bank income, surgeon density)
3. Generate structured research protocol with Lancet Commission 6 indicators
4. Optional: feed into manuscript pipeline (STROBE) or presentation pipeline

# /asf:journal-club

Structured single-article critique (9-section format).

## Usage

```
/asf:journal-club {article or PMID} [--output presentation|critique|both]
```

## Workflow

1. Load article (uploaded PDF or retrieve via PMID)
2. Run 9-section analysis: presenter info, background, design, results, discussion, strengths/limitations, clinical applicability, literature comparison, discussion questions
3. If presentation: author slides using templates/journal-club.md
4. If critique: run peer review engine (5-domain)
5. Generate discussion questions and teaching points

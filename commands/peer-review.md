# /asf:peer-review

Structured manuscript critique using 5-domain review engine.

## Usage

```
/asf:peer-review {manuscript}
```

## Workflow

1. Load manuscript (uploaded file)
2. Auto-detect study type and select reporting checklist
3. Critique across 5 domains: methodology, statistics, reporting, literature, clinical significance
4. Rate findings: Major (must address), Minor (should address), Optional (suggestions)
5. Optional: generate point-by-point revision response template

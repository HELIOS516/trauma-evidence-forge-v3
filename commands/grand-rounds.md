# /asf:grand-rounds

Full evidence-to-presentation pipeline.

## Usage

```
/asf:grand-rounds {topic}
```

## Workflow

1. Run evidence synthesis (Module 1 + domain module)
2. Author slides using assertion-evidence format from templates/
3. Run 5-script Gamma pipeline (scripts/)
4. Validate: 11/11 hard checks must pass
5. Submit to Gamma or build local PPTX

## Output

Gamma-ready presentation with speaker notes, design instructions, and API params.

# /asf:egs-guidelines

EGS condition-specific guideline synthesis and comparison.

## Usage

```
/asf:egs-guidelines {condition} [--societies WSES,SAGES,EAST,AAST]
```

## Workflow

1. Identify EGS condition category (see references/egs-evidence-guide.md)
2. Run domain-specific PubMed search
3. Build severity classification (AAST grading, Hinchey, Tokyo, Atlanta)
4. Produce multi-society guideline comparison matrix
5. Generate operative indications, timing, and antibiotic tables
6. GRADE-rate all evidence

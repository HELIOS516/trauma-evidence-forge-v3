# /asf:critical-care-protocol

SCC evidence synthesis with protocol development.

## Usage

```
/asf:critical-care-protocol {topic} [--societies SCCM,SSC,EAST,ATS]
```

## Workflow

1. Identify SCC topic category (see references/scc-evidence-guide.md)
2. Run domain-specific PubMed search
3. Build landmark trial comparison table
4. Produce guideline comparison matrix
5. Generate protocol summary (agent, dose, route, frequency, duration, monitoring)
6. Include relevant scoring systems
7. GRADE-rate all evidence

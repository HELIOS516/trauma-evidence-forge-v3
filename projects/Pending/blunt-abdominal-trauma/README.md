# Blunt Abdominal Trauma: FAST and Beyond

## Overview
- **Type:** Trauma
- **Duration:** 35 minutes
- **Template:** presentation-medium (18-22 slides)
- **Shelf Exam Weight:** ~10% (abdominal trauma high-yield for ACS shelf)
- **Target Audience:** MS3/MS4 on ACS/Trauma rotation
- **Presenter:** Evan DeCan, MD — UVA Division of Acute Care Surgery

## Learning Objectives
1. Interpret FAST exam findings and integrate results into the trauma algorithm
2. Apply AAST organ injury grading to guide operative vs nonoperative decision-making
3. Determine which blunt abdominal trauma patients require operative intervention vs NOM

## Key Frameworks
- FAST (Focused Assessment with Sonography in Trauma): 4 standard views
- eFAST: extended with bilateral thoracic views
- AAST solid organ injury grading (liver grades I–VI, spleen grades I–V)
- EAST NOM guidelines for blunt solid organ injury

## Core Decision Points
- Imaging vs OR: hemodynamically unstable + positive FAST = OR; stable = CT first
- NOM eligibility: hemodynamic stability, no peritonitis, CT-confirmed grade, IR availability
- NOM failure criteria: hemodynamic deterioration, peritonitis, falling Hgb requiring >2 units
